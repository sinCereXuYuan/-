### 雄霸天下 & 雪飘人间
剧里明说了很多次金刚不坏神功会变成怪物, 但其实是雄霸天下(/雪飘人间). 这两个招式从名字上就很对称 ..

“金刚不坏神功”使用了掌心会有裂痕, (素心说的, 掌心是肉掌背也是肉, 掌心对应着成是非, 掌背对应着神侯)

— 那天晚上 一刀一直发烧 (21)

— (朗儿)这里的井水喝了便肚子疼(25)

— 他(朗儿)的病还未痊愈 身上很热(27)

// 明的是 成是非出事胸口有个大窟窿, 素心会觉得热, 暗的是 归海百炼 要用雄霸天下, 一刀会发热, 暗的是 飘絮要用 雪飘人间 朗儿会出事 ..

一个解释. 飘絮在和柳生但马守对决(第一次真正)用出雪飘人间之后, 装疯是假, 但看不得 兵器/血 却是真的.

雄霸天下和雪飘人间相生相克. 一个会使习武之人看到兵器就想用就会被刀控制, 一个会使习武之人看到兵器就恐惧 .. 所以在蛇岛的那段日子, 也许飘絮是在学习(三分之一的)雄霸天下来以毒攻毒 .. (对比剧集里的, 成是非金刚不坏神功解了神侯的吸星大法寒冰剧毒 ..)

雄霸天下和雪飘人间 还有一个对称之处 .. 归海一刀的成长过程是”绝情绝义绝爱绝亲绝友”, 心里只知道为父报仇, 小飘絮的成长过程是向往着爱情 ..

(另外一个对称的地方)两人都有奇怪的癖好 .. 飘絮喜欢装作雪姬, 一刀恋物

### 你若撒野
// 发如雪 概括了天下第一的角色性格故事, 但偏偏最后要加一句 “你若撒野 今生我把酒奉陪”, 可以理解为, 这是在讲最后成是非和云罗的开心结局, 也可以理解为是 剧情里还有飘絮的撒野暗线 ..

飘絮急着回蛇岛, 也应该是算到, 后面 神侯 和 皇上 要闹翻, 然后天涯会为难, 她也会为难 .. (以及嫁祸一刀的真相可能败露)

(明的是神侯开始并不知道成是非是素心的儿子, 但他的性格决定了他在人前是不会承认自己不知道的, 暗的是神侯并没有下命令杀了海棠, 但是他也不能承认 .. )

### 雪飘人间VS大悲掌(27)
剧里 雪飘人间/雄霸天下/悲天悯人 是同一级别的武功, 在 金刚不坏和吸星大法之上 .. 这三个武功和其他武功的交集比对, 剧里只看到了 霸刀和童子功相平, 吸星大法略在童子功之上, 雄霸天下远在霸刀之上 ..

(会了雄霸天下的一刀报父仇的时候, 三个高手都是自尽的, 并没有迫他使出这一招, 只有后面对决了结的时候用上了, 然后被大悲掌碾压 .. )

其实我更想认定那是飘絮. 飘絮有相夫教子的限制, 但没有利用猪队友或者隐藏兵器赢的不光彩的限制 (柳生但马守最后对决段天涯的时候, 武士精神支配着他不能用隐藏兵器).. 而且(替身?)身型瘦弱, 招式轻盈快落 .. 而且柳生旦马守不会雪飘人间 .. 而且飘絮会因为天涯而不想追杀归海一刀, 尽管找不到的死人栽赃更方便, 柳生旦马守就没这个限制了 ..

(明的是海棠男子装扮行走江湖方便, 暗的是飘絮先用男替身(?), 才能起到反转的效果 .. 海棠男子装扮忽悠的是剧里的角色, 飘絮男替身(?)忽悠的是观众 .. )

(明的是你第一次看很难猜到杀死归海百炼雄霸天下的是一个女人, 暗的是没几个中国人愿意认同, 中土的武林圣僧被东瀛女忍者所杀.)

### "bug" 解释
// 前期段天涯反复回忆的”bug”, 有一个完美的解释 ..

第一遍, 你看到了段天涯和柳生雪姬命运的两个戏剧高峰点.

第二遍, 你看到了整个的故事经过.

// 所以 中后期的剧情

=> 第一遍, 你看到了神侯才是大反派的戏剧性反转.

=> 第二遍, 你看到了神侯是怎么一步步的干掉曹正淳, 怎么谋反.

=> 第"三"遍, 你看到了圆圆飘絮在其中的作用.

本剧的重心不在财力的override, 所以看到万三千随手一张银票就能改变小松的命运, 万三千觉得富贵村值得投入, 就瞬间改变了一个村的命运 .. (也不在传统武力的override上, 所以成是非临战看两眼就会了, 而在 智力/情报/历史 的 override, 所以有很详细的铺垫和consistent, xxxx)

本剧的重心不在除恶惩奸, 所以看到成是非轻松愉快戏弄狗官的剧情 .. (而在”对付” 神侯/柳生但马守/曹正淳 这样的枭雄 .. )

本剧的重心不在严肃的主旋律教育, 所以看到 成是非 各种轻松欢乐, 各种乱用中文成语现代词汇 ..

本剧的重心不在浪漫之情上, 所以恶人峰那个裂谷莫名其妙的出现, 然后一段浪漫的风筝剧情 .. (所以天山雪莲必须要磨碎的设定, 然后一段感人的剧情 .. 所以万三千的莫名其妙的命中注定爱海棠 ..)

本剧的重心不在两军相对上, 所以曹正淳的黑衣箭队, 被人挖到山顶然后团灭 ..

另外飘絮在讲述全盘历史的时候, 说假乌丸假利秀是神侯安排的, 但仔细看剧你会发现, 神侯去调查乌丸利秀的身份时候的自言自语, 他并不知道, 这个bug详见下面写的 ### Die Hard, With a Vengeance
